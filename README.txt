This script generates STD networks as described in https://doi.org/10.1016/j.cor.2021.10567 
and proposed in https://doi.org/10.1007/s11067-017-9345-2.

See the example (AnaheimAux) to understand format requirements. Modify the script at your convenience. 

Have fun!
